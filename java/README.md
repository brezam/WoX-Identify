# Might & Magic IV & V - World of Xeen -- Id Equipment Stats.

## Java

### How to build and run

1. Install the necessary dependencies present in `pom.xml` using maven.
```shell
mvn clean install
```
2. Use the javafx plugin to run

```shell
javafx:run
```
